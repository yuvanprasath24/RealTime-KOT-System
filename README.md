# 🍽️ KOT (Kitchen Order Ticket) System

A web-based application to manage restaurant orders efficiently between waiters and kitchen staff.

## 🚀 Features
- 🧾 Create and manage orders
- 📋 Real-time order tracking
- 🔄 Update order status (Pending → Preparing → Completed)
- 👨‍🍳 Kitchen dashboard view
- 💰 Billing support (optional)

## 🛠️ Tech Stack
- Backend: Spring Boot
- Frontend: React (or mention if different)
- Database: MySQL / PostgreSQL (update accordingly)
- API: RESTful services
